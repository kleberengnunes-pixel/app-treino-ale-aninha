APP ALÊ & ANINHA TREINOS — V2

ARQUIVOS:
- app.py
- requirements.txt

COMO RODAR NO WINDOWS:
1. Extraia este pacote.
2. Abra o CMD dentro da pasta extraída.
3. Execute:
   py -m pip install -r requirements.txt
4. Execute:
   py -m streamlit run app.py

PUBLICAÇÃO NO STREAMLIT CLOUD:
1. Crie um repositório no GitHub.
2. Envie app.py e requirements.txt.
3. Acesse https://share.streamlit.io
4. Clique em Create app.
5. Selecione o repositório, branch main e arquivo app.py.
6. Clique em Deploy.
7. Copie o link gerado e envie para Alê e Aninha.

OBSERVAÇÃO:
O histórico desta V2 fica temporariamente na sessão do navegador.
Para guardar os dados permanentemente, baixe o CSV ao fim do treino.
A V3 pode conectar Supabase ou Google Sheets.
