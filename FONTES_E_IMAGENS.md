# Fontes, imagens e licença

As imagens principais em `assets/componentes/` foram geradas programaticamente como ilustrações didáticas. Elas evitam dependência de imagens com direitos autorais de terceiros.

As imagens em `assets/referencias/` foram geradas a partir de arquivos enviados pelo usuário na conversa. Use apenas como referência interna, salvo se houver autorização/licença para publicação.

Para substituir por imagens reais, recomenda-se usar:

- fotos próprias do laboratório/oficina;
- imagens de fabricantes com autorização formal;
- acervos com licença explícita, como Wikimedia Commons, conferindo a licença de cada arquivo individualmente.

DTCs genéricos OBD-II podem variar por fabricante e aplicação. O simulador usa códigos típicos para fins didáticos.
